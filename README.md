«Червона книга України: Тваринний світ» — це інтерактивний інформаційний довідник, присвячений рідкісним та зникаючим видам тварин нашої країни. Мета проєкту — привернути увагу до проблеми збереження біорізноманіття та надати стислу інформацію про мешканців лісів, степів та водойм, які потребують особливого захисту держави.
